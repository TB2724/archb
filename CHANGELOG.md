# Changelog

## [0.1.0] - 2026-04-19

### Added
- File watcher tracks code changes automatically across your workspace
- AI-powered questions after each coding session (ChatGPT, Claude, and local Ollama)
- Automatic changelog generation: Live Changelog with questions and answers, Technical Changelog and Customer Release Notes
- Q&A log saved live after every answered question
- Raw diff log updated in real-time as you code
- GitHub Push with AI-generated commit message
- Git Settings in sidebar (user.name, user.email, remote, branch)
- AI Settings in sidebar (ChatGPT token, Claude token)
- Snapshot of full project at session start (version backup)
- Clustering of related changes into logical groups before asking questions
- Auto-ask dialog on VS Code restart if unanswered changes exist
- Fallback questions when AI is not reachable
